# ``SwiftHTTPie``

@Metadata {
  @TechnologyRoot
}

SwiftHTTPie bundles the reusable request parser, transport abstraction, and response formatting logic that underpin the SwiftHTTPie CLI. Use it to power command-line tools, embed HTTP workflows into applications, or swap transports for in-process peer execution.
